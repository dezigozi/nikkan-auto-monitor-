設定

監視キーワード

一覧は keywords.json（リポジトリで管理）を読み込んで表示しています。scraper 実行時も同じファイルが使われます。

キーワードの追加・削除は keywords.json を編集して GitHub にプッシュしてください。下の編集はプレビュー用です（保存でダウンロードされるのは config.json 全体のみ）。

※ ログイン・Slack 等は config.json。監視語は keywords.json を編集してください。

データソース（日刊自動車新聞）

netdenjd.com へのログイン情報（config.json に保存）

日付は自動的に付加されます（例：...?date=2026-04-16）

Slack 通知設定

取得・要約完了後に自動投稿

GitHub PAT（手動実行用）

「最新データを取得」ボタンで GitHub Actions を直接トリガーするために使います。

PAT は このブラウザのローカルにのみ保存されます。GitHub には送信しません。
必要な権限: repo（または actions:write）

実行スケジュール

Windows タスクスケジューラで run.bat を登録してください

# タスクスケジューラに登録するコマンド

schtasks /create /tn "AutoNews"

  /tr "C:\path\to\run.bat"

  /sc daily /st 07:00

  /ru SYSTEM

※ 詳細は run.bat 内のコメントを参照してください
